## Run Locally

Clone the project

```bash
  git clone https://github.com/dimonoy/portfolio-website.git
```

Go to the project directory

```bash
  cd portfolio-website
```

Install dependencies

```bash
  python3 -m venv .venv
  source .venv/bin/activate
  pip install .
```

Start the server

```bash
  python manage.py runserver
```
